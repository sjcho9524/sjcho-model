박스 모델


가로 너비, 세로 너비, 내부 여백, 외부 여백 (기본적으로 사각형이다)

1.너비

width = 가로너비
height = 세로너비
auto 브라우저가 너비를 계산 기본값 (요소에 이미 들어있는 속성의 값)
단위 px, em, vw 등 단위 지정

span

<.span>대표적인 인라인요소! 본질적으로 아무것도 나타내지 않는, 콘텐츠 영역을 설정하는 용도.
auto (가로)포함한 콘텐츠 만큼 자동으로 줄어듬!
auto (세로)포함한 콘텐츠 크기만큼 자동으로 줄어듬!

div

<.div> 대표적인 블록요소! 본직적으로 아무것도 나타내지 않는, 콘텐츠 영역을 설정하는 용도.
auto (가로)부모요소의 크기만큼 자동으로 늘어남!
auto (세로)포함한 콘텐츠 크기만큼 자동으로 줄어듬!

요소가 커질 수 있는 최대 가로/세로 너비

max-width, max-height
nono 최대 너비 제한 없음
auto 브라우저가 너비를 계산
단위 px, em, vw 등 단위로 지정

요소가 작아질 수 있는 최소 가로/세로 너비

min-width, min-height
0 최소 너비 제한 없음
auto 브라우저가 너비를 계산
단위 px, em, vw 등 단위로 지정

단위

px 픽셀
% 상대적 백분율
em 요소의 글꼴 크기
rem 루트 요소(html)의 글꼴 크기 루트란? 최상위 요소이다.
vw 뷰포트 가로 너비의 백분율
vh 뷰포트 세로 너비의 백분율

외부여백

margin margin은 음수를 사용할 수 있다.
요소의 외부 여백(공간)을 지정하는 단축 속성

0 외부 여백 없음
auto 브라우저가 여백을 계산 가로(세로) 너비가 있는 요소의 가운데 정렬에 활용!
단위 px,em, vw 등 단위로 지정
% 부모 요소의 가로 너비에 대한 비율로 지정


margin: 10px; top, right, bottom, left 동시에 값이 입력됨
margin: 10px 20px; 앞에 값top, bottom 뒤에 값right, left
margin: 10px 20px 30px 앞에 값top 중간 값right, left 마지막 값 bottom
margin: 10px 20px 30px 40px; top, right, bottm, left 순서대로



요소의 외부 여백을 지정하는 기타 개별 속성들
margin-방향


margin-top
margin-right
margin-bottom
margin-left

내부여백

padding

padding 내부에 여백이 생긴만큼 요소의 크가가 늘어남
요소의 내부 여백을 지정하는 단축 속성

0 내부 여백 없음
단위 px, em, vw 등 단위로 지정
% 부모 요소의 가로 너비에 대한 비율로 지정

padding: 10px; top, right, bottom, left 동시에 값이 입력됨
padding: 10px 20px; 앞에 값top, bottom 뒤에 값right, left
padding: 10px 20px 30px; 앞에 값top 중간 값right, left 마지막 값 bottom
padding: 10px 20px 30px 40px; top, right, bottm, left 순서대로

요소의 외부 여백을 지정하는 기타 개별 속성들
padding-방향

padding-top
padding-right
padding-bottom
padding-left

테두리 선과 색상표현

border

border: 선-두께 선-종류 선-색상;
width style color
요소의 테두리 선을 지정하는 단축 속성
테두리 선이 생긴만큼 요소의 크가가 늘어남
기본값 border: medium none black;
border-width
medium 중간 두께
thin 얇은 두께
thick 두꺼운 두께
단위 px, em % 등 단위로 지정


border-width: top, right, bottom, left;
border-width: (top, bottom) (right, left);
border-width: (top) (right, left) (bottom);
border-width: (top) (right) (bottom) (left);

border-style


요소 테두리 선의 종류
기본값 선 없음
solid 실선(일반 선)
dotted 점선
dashed 파선
double 두 줄 선
groove 홈이 파여있는 모양
ridge 솟은 모양(groove의 반대)
inset 요소 전체가 들어간 모양
outest 요소 전체가 나온 모양


border-style: top, right, bottom, left;
border-style: (top, bottom) (right, left);
border-style: (top) (right, left) (bottom);
border-style: (top) (right) (bottom) (left);

border-color


요소 테두리 선의 색상을 지정하는 단축 속성
기본값 검정색
색상 선의 색상
transparent 투명

색상 표현


색을 사용하는 모든 속성에 적용 가능한 색상 표현
색상 이름 브라우저에서 제공하는 색상 이름 red, tomato, royalblue
Hex색상 코드 16진수 색상 #000, #FFFFFFFF
RGB 빛의 삼원색 rgb(255, 255, 255)
RGBA 빛의 삼원색 + 투명도 rgba(0, 0, 0, 0.5)
HSL 색상, 채도, 명도 hsl(120, 100%, 50%)
HSLA 색상, 채도, 명도 + 투명도 hsla(120, 100%, 50%, 0.3)

요소의 테두리 선을 지정하는 기타 속성들


border-방향
border-방향-속성

border-radius


모서리를 둥글게 깎음
0 둥글게 없음
단위 px, em, vw 등 값으로 단위를 지정

border-raidus:
top, right, bottom, left

box-sizing


요소의 크기 계산 기준을 지정
content-box 요소의 내용(content)으로 크기 계산
border-box 요소의 내용 + padding + border로 크기 계산

overflow

요소의 크기 이상으로 내용이 넘쳤을 때, 보여짐을 제어하는 단축속성
visible 넘친 내용을 그대로 보여줌
hidden 넘친 내용을 잘라냄
scroll 넘친 내용을 잘라냄, 스크롤바 생성 수평 수직 둘 다 무조건 생성
auto 넘친 내용이 있는 경우에만 잘라내고 스크롤바 생성 수평 수직 중 넘친곳만 생성


요소의 크기 이상으로 내용이 넘쳤을 때, 보여짐을 제어하는 개별 속성들
overflow-x x축 가로
overflow-y y축 세로

display

요소의 화면 출력(보여짐)특성
block 상자(레이아웃) 요소 각 요소에 지정되어있는 값
inline 글자 요소 각 요소에 지정되어있는 값
inline-block 글자 + 상자 요소 각 요소에 지정되어있는 값
flex 플렉스 박스 (1차원 레이아웃) 따로 지정해서 사용하는 값
grid 그리드 (2차원 레이아웃) 따로 지정해서 사용하는 값
none 보여짐 특성 없음, 화면에서 사라짐 따로 지정해서 사용하는 값
기타 table, table-row, table-cell 등..

opacity

요소의 투명도
1 불투명
0 투명
0~1 0부터1 사이의 소수점 숫자
