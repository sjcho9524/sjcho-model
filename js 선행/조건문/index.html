조건문

조건의 결과(truthy, falsy)에 따라 다른 코드를 실행하는 구문
if, else

let isShow = true;
let checked = false;

if (isShow) {
console.log('Show!');
}

if (checked) {
console.log('Cecked!');
}

let isShow = true;

if (isShow) {
console.log('Show!');
} else {
console.log('Hide?');
}
