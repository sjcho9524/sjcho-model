변수

데이터를 저장하고 참조(사용)하는 데이터의 이름
var, let const (var)는 가급적 사용하지 않는것이 좋다

재사용이 가능!
변수 선언!

let a = 2;
let b = 5;

console.log(a + b) 7
console.log(a - b) -3
console.log(a * b) 10
console.log(a / b) 0.2

값(데이터)의 재할당 가능!

let a = 12;

console.log(a) 12

a = 999;
console.log(a); 999

값(데이터)의 재할당 불가능!
